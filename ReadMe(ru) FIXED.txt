----Инструкция----
При запуске программы у вас может вылезти ошибка. Вам нужно будет указать путь к .exe файлу вашего софта. 
Для этого вам нужно закрыть окно формы, после чего в трее нажать ПКМ по значку ARDOR и указать путь.

Вы можете закрепить это приложение в панели быстрого запуска. Для этого вам необходимо нажать сочетание клавиш WIN+R и ввести в поле команду: shell:startup, 
после чего откроется папка быстрого запуска. 
В нее вам нужно закинуть ярлык .exe файла моей программы. После чего нажать Ctrl+Shift+Esc и в диспетчере задач выбрать автозагрузку, 
затем найти приложение ARDOR TRAY и включить автозапуск. Приложение в свернутом состояни жрет 9-10мб памяти.

По сути, вы можете использовать данную наработку для сворачивания всех программ в трей, где такая возможность не предусмотрена производителем. 
Для этого вам придется в меню конструктора изменить иконку в строке notifyicon на подходящую, в остальном же, принцип прежний и описан выше.
